Setup Instructions

1

Provide Hosting Access

Share your hosting credentials so I can deploy the app:

• FTP/SFTP: host, username, password
• Domain: recipes.yourdomain.com
• Control Panel URL (if available)
• PHP enabled (for backend)

I'll handle all the deployment and configuration.

2

Google Cloud Setup

Create Google OAuth credentials for Drive access:

1. Go to Google Cloud Console
2. Create project: "Family Recipe Manager"
3. Enable APIs: Drive API, People API
4. Create OAuth 2.0 Web credentials
5. Add authorized origin: https://yourdomain.com
6. Share Client ID with me
3

Start Adding Recipes

Once deployed, you can:

• Send recipe photos via Telegram
• I'll process with OCR and add to database
• Access web app at yourdomain.com/recipes
• Generate meal plans and grocery lists
